## Text Post

You can post text to LinkedIn by using the `linkedin` object inside of your API request.

<CodeGroup tag="POST" label="/v1/publish">

```bash {{ title: 'cURL' }}
curl -X POST https://sotsial.com/v1/publish \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{"target": "<my_connection_id>", "post": {"linkedin": {"text": "my special message"}}}'
```

```js
await fetch('https://sotsial.com/v1/publish', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    target: '<my_connection_id>',
    post: { 
      linkedin: {
        text: 'my special message'
      },
    },
  }),
});
```

```python
import requests

requests.post('https://sotsial.com/v1/publish', 
  headers={
    'Authorization': f'Bearer {token}',
    'Content-Type': 'application/json',
  },
  json={
    target: '<my_connection_id>',
    post: {
      linkedin: {
        text: 'my special message'
      },
    },
  },
)
```

## Image Post

You can post an image to LinkedIn by including `media` inside your `linkedin` object.

<CodeGroup tag="POST" label="/v1/publish">

```bash {{ title: 'cURL' }}
curl -X POST https://sotsial.com/v1/publish \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{"target": "<my_connection_id>", "post": {"linkedin": {"text": "my special message", "media": [{"type": "image", "url": "<image_url>"}]}}}'
```

```js
await fetch('https://sotsial.com/v1/publish', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    target: '<my_connection_id>',

        text: 'my special message',
        media: [
          {
            type: "image",
            url: "<image_url>"
          }
        ]
      },
    },

});
```

import requests
