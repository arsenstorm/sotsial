## Text Post

You can post text to Twitter by using the `twitter` object inside of your API request.

<CodeGroup tag="POST" label="/v1/publish">

```bash {{ title: 'cURL' }}
curl -X POST https://sotsial.com/v1/publish \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{"target": "<my_connection_id>", "post": {"twitter": {"text": "my special message"}}}'
```

```js
await fetch('https://sotsial.com/v1/publish', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    target: '<my_connection_id>',
    post: { 
      twitter: {
        text: 'my special message'
      },
    },
  }),
});
```

```python
import requests

requests.post('https://sotsial.com/v1/publish', 
  headers={
    'Authorization': f'Bearer {token}',
    'Content-Type': 'application/json',
  },
  json={
    target: '<my_connection_id>',
    post: {
      twitter: {
        text: 'my special message'
      },
    },
  },
)
```

</CodeGroup>

<Note>
  Text posts are limited to 280 characters for non-premium Twitter accounts, and 25K for premium accounts.
</Note>
